# 邮见南邮 · 新生校园指南

面向南京邮电大学新生的静态校园指南网站，适配桌面端和手机端。

## 内容

- 仙林校区地图与出行指南
- 社团名录与分类筛选
- 官方认定竞赛目录
- 晨跑、作息与上课时间
- 宿舍片区资料与居住评价
- 校园照片、餐饮和专业指南

## 本地运行

在项目目录启动静态文件服务器：

```powershell
python -m http.server 4174 --bind 127.0.0.1
```

然后访问 <http://127.0.0.1:4174/>。

## 数据说明

本项目不是南京邮电大学官方平台。校内安排、竞赛目录、宿舍分配、交通班次等时效信息，请以学校官方通知为准。
